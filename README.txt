/MSSQL
	/Table - директория с файлами создания и заполнения БД

	/Proc - директория с файлами готовых процедур
		GetChildHid - Процедура получения свободного дочернего узла для переданного объекта
		GetClassAttributes - Процедура получения атрибутов для определенного класса
		DataGenerator - Процедура добавления тестовых данных для каждого уровня иерархии
		SetTempObjectTree - Процедура создания поддерева объектов по заданному объекту во временной таблице
		GetObjectTree - Процедура получения поддерева объектов по заданному объекту
		GetObjectsWithParam - Процедура получения списка объектов в поддереве переданного объекта с параметрами	указанными в п4 тестового задания
		GetObjectAttributes - Процедура получения атрибутов объекта со значениями
		ObjectInsert - Процедура добавления объекта
		ObjectValuesInsert - Процедура добавления значений атрибутов объекта

/С#
	Директория проекта Web API

	API для взаимодействия:
	
	1. /api/AddObject/InsertObject (Пример JSON для вставки приведен ниже)
	Создание объекта определенного класса в определенном месте дерева объектов с возможностью задания значений атрибутов.

	2. /api/TreeObjectList/GetObjectTree?id=3 
	Получение поддерева объектов по заданному объекту (только наименования и иерархия).

	3. /api/MainObject/GetObjectAttributes?id=3
	Получение заданного объекта (полностью, со значениями).

	4. /api/ObjectsWithParam/GetObjectsWithParam?ParentId=3 
	Среди потомков заданного объекта (могут быть объекты разных классов) выбрать список насосов, установленных до сегодняшнего дня, с массой более 10 кг.


	Пример JSON для создания объекта (/api/AddObject/InsertObject)

   "ClassId":1,
   "Name":"через POST",
